Snap-O

A fast, tidy macOS app for Android inspection: capture screenshots and recordings, and inspect network traffic from Android devices and emulators.

Download for macOS View on GitHub

Requires macOS 15+ and adb from Android Platform Tools.

Top-Level Features

  1. 01

    Screen capture

    Grab screenshots and recordings quickly, preview recordings instantly, and scrub frame-by-frame when checking animations or UI flow.

  2. 02

    Drag and drop without saving

    Drop captures directly into pull requests, chat, and docs without the usual save-first friction.

  3. 03

    Multi-device and quick controls

    Works across connected devices and emulators, with direct keyboard shortcuts and Android Studio External Tools integration.

  4. 04

    Network Inspector (Alpha)

    Mirror app traffic into the macOS client, inspect requests and responses, and explore payloads with collapsible JSON pretty printing.
